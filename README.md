# Library Management System

This is a simple console-based Library Management System developed in C++. It allows users to manage books, members, and borrowing functions within a library system.

## Features

- **Book Management**: Add, remove, and track book availability.
- **Member Management**: Register and remove library members with unique IDs.
- **Borrowing & Returning**: Allow members to borrow books, assign due dates, and check for overdue fines.
- **Reporting**: Generate reports of borrowed books and overdue items.

## Prerequisites

To run this project locally, ensure you have the following:

- **C++ Compiler**: GCC or any C++11 compatible compiler
- **Git**: For cloning the repository
